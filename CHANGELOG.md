# Changelog

All notable changes to Selenium-Cucumber Pro are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] — 2026-01-01

### Added

- Generate Selenium Java step definitions from Cucumber `.feature` files
- Smart parameter detection for quoted strings, angle brackets, and integers
- Support for Given, When, Then, And, and But step keywords
- Configurable package name, class name, and base test class
- Framework support for Cucumber, TestNG, and JUnit
- Duplicate step detection with automatic unique method naming
- Try-catch error handling in every generated method
- Contextual Selenium scaffolding based on step type and keyword analysis
- Status bar button for one-click Quick Actions menu
- Keyboard shortcuts: `Ctrl+Shift+G`, `Ctrl+Shift+Q`, `Ctrl+Shift+S`
- Right-click context menu integration for `.feature` files
- Create step definition file directly from the editor
- Side-by-side output preview with Java syntax highlighting
- Clipboard copy on every generation
- Configure Settings dialog without opening VS Code settings
- Shared helper methods (`findElementWithWait`, `isElementDisplayed`) in every generated class
